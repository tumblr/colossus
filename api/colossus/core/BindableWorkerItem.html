BindableWorkerItem

trait BindableWorkerItem extends WorkerItem

BindableWorkItem is the public facing trait which is meant for Tasks and other objects which are Bindable to a Worker. WorkerItem itself is directly extended by ConnectionHandler which has a different and specific lifecycle inside of a Worker.
