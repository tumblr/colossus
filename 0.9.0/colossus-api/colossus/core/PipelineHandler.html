PipelineHandler

Related Docs: object PipelineHandler | package core

class PipelineHandler extends CoreHandler with CoreUpstream with ServerConnectionHandler with ClientConnectionHandler with IdleCheck

The PipelineHandler forms the foundation of all pipeline-based connection handlers. It takes the head and tail of a pipeline and properly directs events to it.
